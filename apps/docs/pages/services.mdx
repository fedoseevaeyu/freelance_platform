# Услуги

Эти маршруты можно использовать для получения и создания услуг.

## Создание услуги

`POST /services`

Тело запроса

```json
{
  "title": "Разработка сайта",
  "description": "Создание адаптивных и мобильных версий сайта с использованием HTML, CSS и JavaScript.",
  "category": "id категории",
  "bannerImage": "url обложки",
  "images": ["url изображения 1", "url изображения 2"],
  "tags": ["id тэга 1", "id тэга 2"],
  "features": [
    {
      "name": "Дизайн",
      "includedIn": ["пакет 1", "пакет 2"]
    },
    {
      "name": "Верстка",
      "includedIn": ["пакет 1", "пакет 2"]
    },
    {
      "name": "SEO оптимизация",
      "includedIn": ["пакет 2"]
    }
  ],
  "packages": [
    {
      "name": "пакет 1",
      "price": 100,
      "description": "описание пакета 1",
      "deliveryDays": 5
    },
   {
      "name": "пакет 2",
      "price": 1000,
      "description": "описание пакета 1",
      "deliveryDays": 10
    }
  ]
}
```

Ответ

```json
{
  "slug": "разработка-сайта-jshfghiu"
}
```

## Получение услуг с использованием имени пользователя

`GET /services/user/:username`

Параметры запроса

- `take` - количество услуг для извлечения (необязательно, по умолчанию равно 10)

Ответ

```json
{
  "services": [
  ],
  "next": 21
}
```

## Получение всех услуг

`GET /services`

Параметры запроса

- `take` - количество услуг для извлечения (необязательно, по умолчанию равно 10)

Ответ

```json
{
  "services": [
  ],
  "next": 21
}
```
